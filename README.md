# shadowrocket_for_windows
A GUI client for Windows, support [Xray core](https://github.com/XTLS/Xray-core) and [v2fly core](https://github.com/v2fly/v2ray-core)

## How to use
- If you are new to this, please download shadowrocket_for_windows.zip from [releases](https://github.com/Pawdroid/shadowrocket_for_windows/releases)
- Run v2rayN.exe

## Requirements  
- [Microsoft .NET 6.0 Desktop Runtime ](https://download.visualstudio.microsoft.com/download/pr/513d13b7-b456-45af-828b-b7b7981ff462/edf44a743b78f8b54a2cec97ce888346/windowsdesktop-runtime-6.0.15-win-x64.exe)
